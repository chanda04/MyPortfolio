# Chand Harjpal — Developer Portfolio

A premium, single-file, responsive portfolio website built for placement drives, internships, and entry-level software developer roles.

---

## ✨ Features

- Dark mode by default, with a light mode toggle (preference saved in `localStorage`)
- Glassmorphism navbar with smooth scroll navigation
- Animated hero section with typing effect, floating tech chips, and mouse-follow gradient glow
- Scroll-reveal animations on every section
- Animated skill progress bars and counters
- Project showcase cards with cover images, feature tags, and tech stack
- Achievements, certifications, and current-status sections
- Contact form with client-side validation
- Scroll progress bar, scroll-to-top button, and page loader
- Fully responsive — mobile, tablet, and desktop
- Respects `prefers-reduced-motion` for accessibility
- No build step required — pure HTML, CSS, and vanilla JavaScript in a single file

---

## 🛠 Tech Stack

- HTML5
- CSS3 (custom properties / design tokens, no framework)
- Vanilla JavaScript (IntersectionObserver for scroll reveals, no libraries)
- Google Fonts — Space Grotesk, Inter, JetBrains Mono

## 🚀 Getting Started

No installation needed.

1. Download or clone this folder.
2. Open `index.html` directly in any browser — or use a local server (e.g. the VS Code "Live Server" extension) for the best experience.

```bash
# Optional: serve locally
npx serve .
```

## 📝 Customization Checklist

Before publishing, replace the following placeholders inside `index.html`:

- [ ] Professional photo (hero section)
- [ ] Resume PDF link (`assets/resume.pdf`)
- [ ] Email, phone number, and location
- [ ] LinkedIn, GitHub, LeetCode, HackerRank URLs
- [ ] Education duration and CGPA for MCA and BCS
- [ ] Certifications (name, platform, completion date)
- [ ] Additional projects, if any

## 🌐 Deployment

This site can be deployed for free on any static hosting platform:

- **Vercel** — drag and drop the folder, or connect a GitHub repo
- **Netlify** — drag and drop the folder into Netlify Drop
- **GitHub Pages** — push to a repo and enable Pages in settings
